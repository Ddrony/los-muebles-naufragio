# Los Muebles del Naufragio 🪑

Website institucional desenvolvido para uma marca de restauro artesanal de mobiliário em Madrid. O projeto foca-se na elegância, na história das peças e na experiência do utilizador para clientes de eventos (casamentos) e decoração de interiores.

##  Funcionalidades
- Interface responsiva (Mobile & Desktop).
- Carrossel dinâmico de exibição de produtos.
- Integração direta com WhatsApp para consultoria de disponibilidade.
- Formulário de contacto funcional.
- Mapa dinâmico para localização da oficina/showroom.

##  Tecnologias Utilizadas
- **HTML5** (Estrutura semântica)
- **CSS3** (Layout Flexbox e Grid, Design Responsivo)
- **JavaScript** (Lógica do menu mobile e carrossel de imagens)
- **Google Fonts & FontAwesome** (Tipografia e ícones)
